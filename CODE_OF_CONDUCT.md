# Code of Conduct

This project is maintained as a friendly community localization effort.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Keep feedback focused on the localization.
- Explain suggestions clearly.
- Be patient with review and updates.

## Not Acceptable

- Harassment, insults, or personal attacks.
- Discriminatory language.
- Spam or unrelated promotion.
- Demanding unpaid work from maintainers or contributors.

## Reporting

If a discussion becomes uncomfortable or unproductive, please use GitHub Issues or repository moderation tools when available.

The goal is simple: improve the Brazilian Portuguese localization together.
