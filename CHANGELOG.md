# Changelog

All notable localization changes are documented here.

Future releases should include notes in both English and Portuguese.

## v1.0.4 - Terminology Fix

### English

- Standardized `Buzzsaw` as `Lâminas` when referring to the Mechatech skill category.
- Fixed the equipment tooltip for Buzzsaw-type skill damage.
- Adjusted the related `[Psionic] buzzsaw` description to use `lâmina` instead of `serra`.

### Português

- Padronizado `Buzzsaw` como `Lâminas` quando o termo se refere à categoria de habilidades Mechatech.
- Corrigido o tooltip de bônus de dano das habilidades de Lâminas.
- Ajustada a descrição da lâmina `[Psiônica]` para evitar o uso incorreto de `serra`.

## v1.0.3 - UI Polish

### English

- Shortened tutorial titles and descriptions that could overflow narrow UI panels.
- Fixed the Combat Guide Divine Technology section for better readability.
- Replaced the long storage title with a shorter PT-BR UI label: `Espaços do Armazém`.
- Rebuilt optional patch only; original game files remain untouched.

### Português

- Encurtados títulos e descrições de tutorial com risco de estourar painéis estreitos da interface.
- Ajustada a seção Tecnologia Divina do Guia de Combate para melhorar a leitura.
- Substituído um título longo de armazenamento por `Espaços do Armazém`.
- Patch opcional reconstruído sem alterar arquivos originais do jogo.

## v1.0 - Initial Release

### English

- Complete Brazilian Portuguese localization source included.
- Translation reviewed for terminology consistency, naturalness, UI readability, and gameplay clarity.
- Major text categories covered: menus, UI, equipment, abilities, events, dialogue, and update notes.
- Glossary and translation notes included for review.
- Optional standalone patch included for testing.

### Português

- Localização PT-BR completa incluída.
- Tradução revisada quanto a consistência terminológica, naturalidade, legibilidade de interface e clareza de gameplay.
- Categorias principais cobertas: menus, UI, equipamentos, habilidades, eventos, diálogos e notas de atualização.
- Glossário e notas de tradução incluídos para consulta.
- Patch autônomo opcional incluído para teste.
