# Changelog

## Version 1.0

Initial professional localization package prepared for developer and publisher evaluation.

### Localization

- Completed Brazilian Portuguese localization source included.
- Translation reviewed for terminology consistency, naturalness, UI readability, and gameplay clarity.
- Major text categories covered: menus, UI, equipment, abilities, events, dialogue, and update notes.
- Glossary and translation notes included for review.

### Technical Package

- Source CSV files included for evaluation.
- Optional standalone patch included for testing.
- Original game files remain untouched.
- Patch can be removed by deleting `RIP_PTBR_P.pak` from the game's `Paks` folder.

### Notes

This release is a voluntary, non-commercial community contribution. It is provided free of charge and carries no request for payment, royalties, employment, or financial compensation.
