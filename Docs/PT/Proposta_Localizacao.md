# Proposta de Localização PT-BR

## Reincarnation Insurance Program

Este documento apresenta a localização brasileira de **Reincarnation Insurance Program** como uma contribuição gratuita da comunidade.

O projeto foi criado para tornar o jogo mais acessível a jogadores brasileiros e, ao mesmo tempo, facilitar uma possível avaliação técnica por WarmCore Studio e 2P Games.

## Objetivo

O objetivo da localização é fazer com que o jogo soe natural em português do Brasil, mantendo o tom original, a clareza da interface e a consistência dos sistemas de combate, equipamentos e progressão.

A tradução cobre:

- menus e interface;
- textos de combate;
- equipamentos, armas e atributos;
- habilidades e tecnologias divinas;
- eventos e mensagens de gameplay;
- diálogos;
- notas de atualização;
- textos de tutorial e interface.

## Qualidade da Localização

O projeto passou por revisão de linguagem, revisão terminológica, testes em jogo e ajustes de interface. O foco foi evitar tradução literal e priorizar uma experiência fluida para jogadores brasileiros.

Algumas decisões importantes:

- termos de sistemas foram padronizados em glossário;
- textos longos de interface foram encurtados quando necessário;
- tags, placeholders e formatação foram preservados;
- problemas encontrados durante gameplay foram corrigidos em versões posteriores.

## Estrutura Técnica

O patch opcional é distribuído como `RIP_PTBR_P.pak`. Ele funciona como um arquivo separado e não modifica o pacote original do jogo.

Para avaliação técnica, o repositório também inclui:

- CSV com o texto original em inglês;
- CSV com a localização PT-BR;
- glossário;
- notas de tradução;
- documentação de problemas conhecidos;
- patch opcional para teste em jogo.

## Contribuição Gratuita

Esta localização é oferecida gratuitamente. Não há pedido de pagamento, royalties, emprego ou compensação financeira.

Se a localização for útil para uma integração oficial, o crédito pode ser atribuído ao usuário GitHub **vigarizta**.

## Manutenção Comunitária

Além de servir como proposta técnica, este repositório também funciona como ponto central da comunidade para reportar erros, sugerir melhorias e acompanhar novas versões da localização.
