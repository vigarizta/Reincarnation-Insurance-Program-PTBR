# Documentação em Português

Este é o espaço principal da documentação em português da localização PT-BR de **Reincarnation Insurance Program**.

O objetivo do projeto é manter uma tradução brasileira natural, consistente e fácil de atualizar, tanto para jogadores quanto para possíveis revisões oficiais pelos desenvolvedores.

## Para Jogadores

Baixe a versão mais recente em [Releases](https://github.com/vigarizta/Reincarnation-Insurance-Program-PTBR/releases).

Para instalar:

1. Baixe o pacote da Release.
2. Abra a pasta `Optional_Patch`.
3. Copie `RIP_PTBR_P.pak` para a pasta `AKA_RIP/Content/Paks` do jogo.
4. Abra o jogo normalmente.

O patch não substitui arquivos originais do jogo. Para remover a tradução, apague apenas `RIP_PTBR_P.pak`.

## Como Reportar Erros

Encontrou texto em inglês, frase estranha, texto cortado ou problema de interface? Abra uma Issue:

https://github.com/vigarizta/Reincarnation-Insurance-Program-PTBR/issues

Sempre que possível, inclua:
