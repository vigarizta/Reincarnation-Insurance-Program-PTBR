# Glossário PT-BR

Este glossário reúne decisões terminológicas importantes da localização brasileira.

## Termos Principais

| Inglês | PT-BR |
| --- | --- |
| Ailment Resist | Resistência a Aflições |
| HP | Vida |
| Helmet | Capacete |
| Gloves | Luvas |
| Wristband | Bracelete |
| Goggles | Óculos |
| Belt | Cinto |
| Grenade | Granada |
| SMG | SMG |
| Shotgun | Espingarda |
| Rocket Launcher | Lança-Foguetes |
| Sniper Rifle | Rifle de Precisão |
| Compound Bow | Arco Composto |
| Heavy | Pesado |
| Light | Leve |
| Drop Rate | Chance de Obtenção |
| Buzzsaw | Lâminas, quando usado como categoria de habilidade Mechatech |

## Observações

`Drop` não possui tradução fixa em todos os contextos. A tradução depende do uso:

- `Drop Rate` -> `Chance de Obtenção`;
- item obtido de inimigo pode exigir outra solução;
- texto de interface pode pedir uma frase mais curta.

## Categorias de Texto

| Namespace | Uso |
| --- | --- |
| GeneralText | Menus, interface, tutoriais e mensagens gerais |
| EquipmentText | Equipamentos, armas, atributos e descrições |
| DivineTechText | Tecnologias divinas, habilidades e bônus |
| EventText | Eventos e efeitos de gameplay |
| ScriptText | Falas, narrativa e textos de personagem |
| UpdateNotesText | Notas de atualização |

## Regra Geral

Quando houver dúvida, a prioridade é:

1. clareza para o jogador;
2. consistência com o glossário;
3. naturalidade em português brasileiro;
4. respeito ao contexto do jogo.
