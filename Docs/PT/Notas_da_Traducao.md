# Notas da Tradução

## Abordagem

A localização foi feita com foco em português brasileiro natural, leitura rápida durante o gameplay e consistência entre menus, equipamentos, habilidades e diálogos.

O objetivo não foi traduzir palavra por palavra, mas adaptar o texto para soar como uma versão oficial do jogo em PT-BR.

## Terminologia

Os termos recorrentes foram padronizados em glossário. Isso inclui atributos, tipos de equipamento, categorias de habilidade, efeitos, raridades e mensagens de interface.

Exemplos de decisões:

- `HP` foi localizado como `Vida`;
- `Drop Rate` foi localizado como `Chance de Obtenção`;
- `Buzzsaw`, quando usado como categoria de habilidade Mechatech, foi padronizado como `Lâminas`;
- nomes próprios e alguns termos de facção foram preservados quando faziam parte da identidade do jogo.

## Interface

Durante os testes, alguns textos foram encurtados para evitar estouro de layout, cortes e quebras ruins de linha.

A prioridade foi manter clareza sem perder significado.

## Teste em Jogo

A tradução foi testada dentro do jogo usando um patch separado. Os testes ajudaram a encontrar:

- textos que ainda apareciam em inglês;
- problemas de UI;
- termos inconsistentes;
- frases longas demais;
- descrições de equipamento com leitura ruim.

## Formatação

Foram preservados:

- placeholders;
- tags RichText;
- números;
- quebras de linha importantes;
- estrutura CSV;
- IDs e chaves internas.

## Futuras Revisões

Mesmo com a versão atual validada, novos problemas podem aparecer conforme mais jogadores testem classes, itens e telas diferentes. Esses casos devem ser reportados via GitHub Issues.
