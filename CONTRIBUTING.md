# Contributing

Thank you for helping improve the Brazilian Portuguese localization of **Reincarnation Insurance Program**.

## How to Help

You can contribute by:

- opening Issues for untranslated text;
- reporting English strings that still appear in-game;
- reporting UI overflow or clipped text;
- suggesting more natural Brazilian Portuguese phrasing;
- pointing out terminology inconsistencies;
- reporting formatting problems, broken tags, or incorrect line breaks;
- reviewing glossary decisions.

## Reporting a Translation Issue

Please use GitHub Issues only:

https://github.com/vigarizta/Reincarnation-Insurance-Program-PTBR/issues

Include:

- a screenshot;
- where it occurred;
- the current text;
- your suggestion, if you have one.

## Before Suggesting a Change

Please check:

- whether the term already appears in the glossary;
- whether the suggested wording fits the UI space;
- whether the change keeps the tone natural for Brazilian players.

## Pull Requests

Pull requests are welcome for documentation improvements and clear localization fixes.

For translation changes, please explain the context and, when possible, include a screenshot from the game.

## Language

You may open Issues in Portuguese or English.
