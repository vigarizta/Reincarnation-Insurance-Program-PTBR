# Reincarnation Insurance Program (R.I.P.) - Brazilian Portuguese Localization

## Project Overview

This package contains a complete Brazilian Portuguese localization project for **Reincarnation Insurance Program (R.I.P.)**. It was prepared as a voluntary, non-commercial community contribution for review by WarmCore Studio and 2P Games.

The localization was created after completing and enjoying the game. The goal was to help Brazilian Portuguese players experience the game with clear, natural, and consistent language while preserving the original tone and identity of the work.

## About the Game

Reincarnation Insurance Program (R.I.P.) is an action-focused game with equipment, progression systems, combat terminology, interface text, update notes, and narrative dialogue. Because the game contains many systems-driven terms, a consistent terminology base was treated as a central part of the localization process.

## Why This Localization Was Created

This project was completed to make the game more accessible to Brazilian Portuguese players. It is offered freely, with no expectation of payment, employment, royalties, or any other form of financial compensation.

Project maintained by GitHub user **vigarizta**.

Questions, feedback, or requests can be sent through the project's GitHub Issues page:

## Review and Quality Process

The localization was prepared with the following goals:

- Natural Brazilian Portuguese suitable for a commercial game release.
- Consistent terminology across UI, equipment, abilities, events, and dialogue.
- Preservation of placeholders, RichText tags, formatting, and technical structure.
- Manual review of translated content.
- In-game testing through a standalone patch.
- Additional checks for residual source-language strings in visible UI.

## Included Materials

This package includes:

- Localization source files used during the project.
- A glossary of standardized terms.
- Translation and review notes.
- A status summary for evaluation.
- An optional standalone patch build for testing.
- Placeholder folders for screenshots that can be added later.

## Technical Note

The optional patch is provided only as a demonstration and testing build. It does not overwrite original game files. The original game package remains untouched, and the localization can be removed by deleting the standalone patch file.

## Free Community Contribution

This localization is provided free of charge as a voluntary community contribution. The included source files are intended to make technical evaluation easier for WarmCore Studio and 2P Games.
